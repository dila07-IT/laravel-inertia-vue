# Laravel Inertia Vue Starter 🚀

A Laravel package to quickly install and configure Inertia.js with Vue 3.

---

## ✨ Features

- Install Inertia.js Laravel adapter
- Setup Vue 3 with Inertia
- Auto install npm dependencies
- Safe file handling with overwrite confirmation
- `--force` support for full overwrite
- Generates setup guide when files are skipped
- Developer-friendly and easy to use

---

## ⚡ Installation

```bash
composer require dilansabah/laravel-inertia-vue

php artisan inertia-vue:install

npm run build
npm run dev
